Query syntax reference

Logic

  • = equal
  • ~ regexp
  • ~= or equal
  • ~+ or regexp
  • , separator

Examples:

  • A=1,B=2 → A==1 && B==2
  • A=1,A=2 → A==1 || A==2
  • A~ab → A==/ab/
  • A=1,B+=2 → A==1 || B==2
  • A=1,B=2,C+=3 → (A==1 && B==2)||C==3
